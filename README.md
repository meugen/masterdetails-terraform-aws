# Service to process Master/Details resources

### Consists of two repositories:
* [masterdetails-service](https://github.com/meugen/masterdetails-service) - main service implementation
* [masterdetails-compose](https://github.com/meugen/masterdetails-compose) - docker compose configuration to service up and running
* masterdetails-terraform-aws - terraform implementation to apply infrastructure at AWS **You are here**

### Uses technologies:
* Spring Boot 3.5.7
* Spring Validations
* Spring Data JPA
* Hibernate ORM
* PostgreSQL
* Flyway
* Redis
* MapStruct
* Docker
* Docker Compose
* Terraform
* AWS / AWS SDK

### Can be tested at [masterdetails](https://9dp5zjmu4q.eu-central-1.awsapprunner.com/)
